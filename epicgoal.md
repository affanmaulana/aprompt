# epic_goal.md

# APROMPT v2

## Infinite Architectural Prompt Intelligence System

### Epic Name

Dynamic Expandable Carriage Architecture

### Branch

```bash
git checkout -b fitur/dynamic-carriage-system
```

---

# PROJECT MISSION

Transform Aprompt from a:

```text
Prompt Generator
```

into a:

```text
Prompt Intelligence Operating System
```

capable of producing architectural visualization prompts at professional visualization studio level comparable to:

* Hayes Davidson
* MIR
* Kunkun Visual
* Brick Visual
* Beauty and the Bit
* Luxigon
* DBOX

The generated prompt must be capable of describing architectural imagery with extremely high fidelity and semantic precision while remaining completely client-side and zero-latency.

The current 9-carriage architecture is no longer sufficient.

The system must evolve into a fully dynamic carriage architecture with no fixed upper limit.

---

# CORE OBJECTIVE

Current architecture:

```javascript
const schema = [
  carriage1,
  carriage2,
  carriage3,
  ...
  carriage9
]
```

must be completely redesigned.

The application must no longer think in terms of:

```text
9 carriages
```

or

```text
fixed prompt sections
```

Instead:

```text
Prompt = Collection of Dynamic Semantic Modules
```

where every module is independently expandable.

---

# NEW SYSTEM PHILOSOPHY

The system should mimic how world-class architectural visualization directors think.

Not:

```text

Environment
```

```text

Composition
Camera
Architecture
Material
Atmosphere
Weather

```

Each category becomes independently expandable.

# DYNAMIC CARRIAGE FRAMEWORK

Replace static schema with:

```javascript
{
  id,
  title,
  category,
  group,
  order,
  priority,
  template,
  options,
  enabled,
  expandable
}
```

---

# DYNAMIC INJECTION REQUIREMENT

The application must support:

```text
Dynamic Carriage Injection
```

New carriage definitions must be injectable directly through schema files.

Example:

```javascript
{
  id: "weather_micro_condition",
  title: "Micro Weather Condition"
}
```

Once added:

* UI appears automatically
* localStorage works automatically
* prompt generator works automatically
* sanitization works automatically
* copy button works automatically

without touching:

```text
App.jsx
LeftPanel.jsx
RightPanel.jsx
CarriageInput.jsx
```
